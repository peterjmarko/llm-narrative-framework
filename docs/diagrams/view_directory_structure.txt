## Directory Structure

This logical hierarchy is reflected in the physical layout of the repository:

llm-narrative-framework/
└── README.md                       # High-level overview and guidepost.
└── CONTRIBUTING.md                 # Guidelines for contributors.
└── pyproject.toml                  # Project dependencies and metadata (PDM).
└── .gitignore                      # Specifies untracked files to ignore.
│
└── .env                            # Stores private API keys (not committed).
└── config.ini                      # Main configuration for experiments.
│
└── prepare_data.ps1                # ENTRY POINT: Run the data preparation pipeline.
└── new_experiment.ps1              # ENTRY POINT: Create a new experiment.
└── audit_experiment.ps1            # ENTRY POINT: Check an experiment's status.
└── fix_experiment.ps1              # ENTRY POINT: Fix/update an experiment.
└── compile_study.ps1               # ENTRY POINT: Compile and analyze a study.
└── audit_study.ps1                 # ENTRY POINT: Check a study's status.
│
└── src/                            # All Python source code for the project.
    └── experiment_manager.py
    └── replication_manager.py
    └── ...                         # (and ~30 other scripts)
│
└── data/                           # Source data, artifacts, and databases.
    └── foundational_assets/
        └── country_codes.csv
    └── sources/
        └── adb_raw_export.txt
    └── intermediate/
        └── adb_eligible_candidates.txt
    └── processed/
        └── subject_db.csv
    └── reports/
        └── adb_validation_report.csv
    └── backup/
        └── backup_file_example.zip
    └── base_query.txt
    └── personalities_db.txt
│
└── output/                         # Generated experiments and analysis reports.
    └── new_experiments/
        └── <Experiment_Name>/
            └── <Replication_Name>/
                └── <Trial_Name>/
                └── config.ini.archived # Config snapshot for the run.
                └── replication_report.txt
            └── EXPERIMENT_results.csv # AGGREGATE: Summary of all replications.
    └── studies/
        └── <Study_Name>/
            └── <Experiment_Name_A>/
            └── <Experiment_Name_B>/
            └── anova/
            └── STUDY_results.csv   # AGGREGATE: Summary of all experiments.
│
└── docs/                           # Project documentation and diagram sources.
    └── DOCUMENTATION.md            # The comprehensive Framework Manual.
    └── article_supplementary_material.md # The Replication Guide for the paper.
    └── diagrams/                   # Source files for diagrams (.mmd, .txt)
        └── arch_project_overview.mmd
    └── images/                     # Rendered images (.png) for documentation
    └── ...                         # (and other documentation files)
│
└── scripts/                        # Utility scripts for maintenance, analysis, and builds.
    └── build/
        └── build_docs.py
    └── maintenance/
    └── ...                         # (and other utility subdirectories)
│
└── tests/                          # Unit and integration tests.
    └── assets/
    └── data_preparation/
        └── test_create_subject_db.py
    └── experiment_lifecycle/
        └── test_experiment_manager.py
    └── ...                         # (and other test subdirectories)
